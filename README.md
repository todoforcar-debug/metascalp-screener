# 🚀 MetaScalp Screener

Простой локальный крипто-скринер для просмотра монет со всех бирж, подключённых в MetaScalp.

## ✨ Особенности

✅ **Автоматическое обнаружение** - находит все биржи, подключённые в MetaScalp  
✅ **Выбор биржи** - просмотр только монет выбранной биржи  
✅ **Реальные данные** - цена, объём за 24ч, изменение за 24ч  
✅ **Поиск и фильтр** - поиск по названию монеты и фильтр по минимальному объёму  
✅ **Локально** - работает только на твоём ПК, никаких облаков  
✅ **Просто** - всего один файл, легко использовать  

## 📋 Требования

- **MetaScalp** запущен на твоём ПК с подключённой хотя бы одной биржей
- **Python 3.7+** установлен
- **pip** (обычно идёт вместе с Python)

## 🚀 Быстрый старт

### 1. Клонируй репозиторий

```bash
git clone https://github.com/todoforcar-debug/metascalp-screener.git
cd metascalp-screener
```

Или просто скачай `screener.py` отсюда.

### 2. Установи зависимости

```bash
pip install -r requirements.txt
```

Или просто:

```bash
pip install metascalp
```

### 3. Запусти скринер

```bash
python screener.py
```

## 📖 Как использовать

1. **Убедись что MetaScalp запущен** на твоём ПК с подключёнными биржами
2. **Запусти скринер**: `python screener.py`
3. **Выбери биржу** из списка (введи номер)
4. **Установи фильтры** (опционально):
   - Минимальный объём (USD)
   - Поиск по названию монеты
5. **Смотри результаты!**

## ❓ Решение проблем

### ❌ "MetaScalp not found" / "Connection refused"

**Решение:**
1. Убедись что **MetaScalp запущен** на твоём ПК
2. Убедись что **есть подключённая биржа** (хотя бы одна)
3. Проверь что **никакой файервол** не блокирует localhost
4. Перезагрузи MetaScalp и попробуй снова

### ❌ "ModuleNotFoundError: No module named 'metascalp'"

**Решение:**
```bash
pip install metascalp
```

### ❌ "Python not recognized"

**Решение:**
1. Переустанови Python с https://python.org
2. **Важно**: при установке выбери "Add Python to PATH"

## 🔧 Технические детали

- Использует официальный **MetaScalp SDK** для Python
- Подключается через **REST API** на localhost
- Никаких внешних сервисов — всё локально
- Асинхронные запросы для быстрой загрузки данных
