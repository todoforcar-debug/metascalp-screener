# 🚀 MetaScalp Screener

Простой локальный крипто-скринер для просмотра монет со всех бирж, подключённых в MetaScalp.

## 🎯 Две версии скринера

### 1. **screener_simple.py** ⭐ (РЕКОМЕНДУЕТСЯ)
**БЕЗ ЗАВИСИМОСТЕЙ** — работает с встроенными библиотеками Python!

```bash
python screener_simple.py
```

✅ Просто скачай и запусти  
✅ Никаких `pip install`  
✅ Работает с Python 3.7+  

### 2. **screener.py**
Оригинальная версия (требует `pip install metascalp`)

```bash
pip install metascalp
python screener.py
```

---

## 💡 Особенности

✅ **Автоматическое обнаружение** - находит все биржи, подключённые в MetaScalp  
✅ **Выбор биржи** - просмотр только монет выбранной биржи  
✅ **Реальные данные** - цена, объём за 24ч, изменение за 24ч  
✅ **Поиск и фильтр** - поиск по названию монеты и фильтр по минимальному объёму  
✅ **Локально** - работает только на твоём ПК, никаких облаков  
✅ **Просто** - всего один файл, легко использовать  

## 📋 Требования

- **MetaScalp** запущен на твоём ПК с подключённой хотя бы одной биржей
- **Python 3.7+** установлен

## 🚀 Быстрый старт (screener_simple.py)

### 1. Скачай файл
Просто скачай `screener_simple.py` отсюда на свой ПК

### 2. Убедись что MetaScalp запущен
Запусти MetaScalp на твоём ПК с подключёнными биржами

### 3. Запусти скринер
Открой PowerShell/CMD в папке со скринером и выполни:

```bash
python screener_simple.py
```

### 4. Выбери биржу и смотри монеты!

## ⚙️ Технические детали

### screener_simple.py (облегчённый)
- Использует только встроенные модули Python: `asyncio`, `json`, `urllib`

## 🆘 Решение проблем

### ❌ "MetaScalp не найден на портах 17845-17855"

**Решение:**
1. Убедись что **MetaScalp запущен** на твоём ПК
2. Убедись что **добавлена хотя бы одна биржа** в MetaScalp
3. Если используешь VPN/Proxy — отключи
4. Перезагрузи MetaScalp и попробуй снова

### ❌ "Python not recognized"

**Решение:**
1. Установи Python с https://python.org
2. **Важно**: при установке выбери "Add Python to PATH"
3. Перезагрузи PowerShell

### ❌ "No module named 'metascalp'" (только для screener.py)

**Решение:**
```bash
pip install metascalp
```

Или просто используй `screener_simple.py` — там не нужна эта библиотека!
